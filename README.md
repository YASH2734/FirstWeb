# 🎮 Game Sale Website

A simple web-based game sale platform developed using HTML, CSS, JavaScript, SQL Database, and Apache Server. The website allows users to browse games, view details, and interact with game-related content through a user-friendly interface.

## 🚀 Features

- Browse available games
- View game details and pricing
- Search functionality
- Database integration for storing game information
- Responsive user interface
- Apache server deployment

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend & Database
- SQL Database (MySQL)
- Apache Server

## ⚙️ Installation

### Prerequisites
- Apache Server (XAMPP/WAMP)
- MySQL Database

### Setup Steps

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Move the project folder to the Apache server directory:

- XAMPP: `htdocs/`
- WAMP: `www/`

3. Start:
   - Apache Server
   - MySQL Server

4. Import the SQL database:
   - Open phpMyAdmin
   - Create a new database
   - Import `game_store.sql`

5. Open your browser:

```
http://localhost/Game-Sale-Website
```
